# frontend-workflow-with-gulp-jade---livereload

Learning web design with gulp

This project is just to start my learning about coding in web design, the code used in this project is by Maximilian Schmitt and you can find it at https://www.youtube.com/playlist?list=PLRk95HPmOM6PN-G1xyKj9q6ap_dc9Yckm, I just followed the tutorial, and i took the opportunity to update some code that wasn't working, at the time, thanks in advance for the teachings, it was interesting to say the least.
Note: to open the web server with "gulp.task connect", you have to open it manually by doing (ctrl+click) http://localhost:8080 in the terminal while running the task to open the server in the browser, not yet I found the code to open the browser automatically.
